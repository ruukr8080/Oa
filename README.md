# 향후 계획

**협업툴 Oa**를 첫번째 주요 상품으로.
향후 대규모 커뮤니티 집중 포털사이트를 개발하고 그 포털사이트의 첫번째 주력 서비스로 이용할겁니다.

포털사이트는 **지식**과 **블로그**를 합친 **지로그** 라는 이명을 가진 **오아시스 3.0**입니다.
